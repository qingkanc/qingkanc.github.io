Homebrew常用命令

API

  • 搜索
    brew search xxx
  • 安装
    brew install xxx
  • 卸载
    brew uninstall xxx
  • 查询
    brew info xxx 查看具体的信息及依赖关系当前版本注意事项等
  • 更新
    brew update
  • 检测新版本
    brew outdated 会列出所有有新版本的程序
  • 升级
    brew upgrade 升级所有 当然也可以指定升级
    brew upgrade xxx指定的升级的程序名
  • 清理
    brew cleanup 清理不需要的版本及其安装缓存
  • 服务停止/启动
    服务.server stop/start
